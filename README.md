# Newton
Newton is an app that keeps track of many tasks that teachers need to do on a daily basis.
* Motivates students with digital stickers and work streaks
* Tracks time spent on task
* Differentiates math instruction
* Scrapes data from google sheets, as well as our math and reading websites.
* Displays a morning message
* Fun teacher tools like auto-group-maker, random student picker, focus music, timers, and more
* Logs in students to math and reading websites
* Assigns math tasks
* Tracks reading growth 
* Attendance alerts
* Monitors Chronic Absenteeism
* Automates weekly progress letters to send to parents
